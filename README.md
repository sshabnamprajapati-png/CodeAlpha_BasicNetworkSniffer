# CodeAlpha Basic Network Sniffer

A Python-based network sniffer tool that captures and analyzes live network packets using Scapy.  
This project is developed as part of the CodeAlpha internship program.

---

## Features
- Captures live network packets
- Displays IP, TCP, UDP information
- Basic traffic monitoring
- Stores captured data in log file

---

## Files Included
- sniffer.py → Main Python script
- log.txt → Captured network traffic logs
- Screenshot.png → Output proof (optional)

---

## Tools Used
- Python
- Scapy

---

## Purpose
This project demonstrates basic network traffic analysis for cybersecurity learning and internship training (CodeAlpha).
